## Getting Started

### Option 1: Open Directly
Simply open `index.html` in your web browser.

### Option 2: Local Server (Recommended)
For the best experience, use a local server:

**Using Python:**
```bash
# Python 3
python -m http.server 8000

# Then open: http://localhost:8000
```

**Using Node.js:**
```bash
npx http-server -p 8000

# Then open: http://localhost:8000
```

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #ff0844;      /* Main brand color */
    --secondary: #00d9ff;    /* Accent color */
    --dark: #0a0e27;        /* Background */
}
```

### Social Links
Update the social media links in `index.html`:
- Twitter/X
- Telegram
- Discord
- Instagram

### Contract Address
Add your contract address to the "Buy $CC" buttons once deployed.

### Tokenomics
Adjust the distribution percentages and bars in the tokenomics section.

## Deployment

### GitHub Pages
1. Push to GitHub
2. Go to Settings > Pages
3. Select main branch
4. Your site will be live at `username.github.io/repo-name`

### Netlify
1. Drag and drop the folder to Netlify
2. Your site is live instantly!

### Custom Domain
Point your domain's DNS to your hosting provider.
